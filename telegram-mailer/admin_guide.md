# Руководство администратора

## Первый запуск
1. Установите Docker и Docker Compose.
2. Скопируйте .env.example в .env, укажите BOT_TOKEN, SUPERADMIN_ID, ENCRYPTION_KEY.
3. Запустите: docker-compose up -d
4. Напишите /start вашему боту.

## Управление аккаунтами
- /add_account – добавить аккаунт (API ID, API Hash, телефон, код, пароль)
- Аккаунты можно проверять на SpamBlock и FloodWait.

## Создание рассылки
- /new_campaign – пошаговый мастер: название, шаблон, аккаунты, группы, интервалы.

## Отчёты
- В меню "Отчёты" выберите рассылку, бот сформирует два Excel-файла.

## Роли
- Super Admin задаётся в .env, может добавлять/удалять админов.
- Админы имеют доступ ко всем функциям, кроме управления админами.
